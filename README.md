# Planeamento Financeiro em Python para Vercel

Estrutura:
- `api/backend.py` — API em Python para dados e cálculos.
- `api/frontend.py` — Frontend em HTML/CSS/JS servido por Python.
- `vercel.json` — Rotas para deployment na Vercel.
- `requirements.txt` — vazio porque usa apenas bibliotecas padrão do Python.

## Deploy
1. Envia estes ficheiros para um repositório GitHub.
2. Importa o repositório na Vercel.
3. Faz deploy.
4. Abre a rota `/`.
